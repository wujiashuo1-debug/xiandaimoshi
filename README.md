# 软件工程刷题网站 V2｜手机优先版

这是一个纯静态刷题网站，不需要后端服务器。题库来自上传的中文软件工程习题册。

## 主要功能

- 手机优先布局：底部导航、单题卡片、大按钮、少滑动。
- 首页章节卡片：按章节查看题量、掌握数和正确率。
- 学习模式：一题一屏，可展开答案、标记掌握、加入错题、收藏。
- 刷题模式：支持全部章节、指定章节、只刷错题。
- 智能顺序：优先刷错题、未练题、薄弱题。
- 错题本：做错自动加入，做对自动移出。
- 进度统计：总进度、章节进度、正确率、已掌握数量。
- 本地保存：学习记录保存在当前浏览器的 localStorage。
- 导出/导入：可备份学习记录 JSON。
- PWA：部署到 HTTPS 后，可添加到手机主屏幕，并支持离线缓存。

## 本地使用

解压后双击 `index.html` 可以打开。

但手机端更推荐部署到网页平台，例如：

- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages

部署后用手机浏览器打开网址，再选择“添加到主屏幕”。

## 文件说明

- `index.html`：网站入口
- `style.css`：手机优先样式
- `app.js`：刷题逻辑
- `questions.js`：题库数据
- `manifest.webmanifest`：PWA 配置
- `sw.js`：离线缓存
- `assets/software_engineering_question_book.pdf`：原始 PDF 对照
