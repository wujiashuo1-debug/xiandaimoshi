# 软件工程刷题网站 V3

这是 V3 双端版：

- 电脑端：保留 V1 的桌面逻辑，左侧章节导航，右侧学习 / 刷题 / 错题 / 进度四个页面。
- 手机端：独立移动端界面，不是简单把电脑网页缩小；按章节入口组织，一次只看一道题，底部四栏导航，刷题按钮固定在底部。

## 使用方式

本地打开：

1. 解压压缩包。
2. 双击 `index.html`。

部署到 GitHub Pages：

1. 新建仓库。
2. 上传本文件夹里面的所有内容，不要上传外层文件夹。
3. 仓库根目录必须直接看到：
   - `index.html`
   - `app.js`
   - `style.css`
   - `questions.js`
   - `assets/`
4. Settings → Pages → Deploy from a branch → main → /root。

## 注意

学习记录保存在浏览器 localStorage 中。电脑和手机是不同浏览器，不会自动同步。可以使用“导出学习记录 / 导入学习记录”迁移。
